gravatar-image
==============================================================================

Easily render gravatar images.

Compatibility
------------------------------------------------------------------------------

* Ember.js v3.4 or above
* Ember CLI v2.13 or above
* Node.js v8 or above


Installation
------------------------------------------------------------------------------

```
ember install gravatar-image
```


Usage
------------------------------------------------------------------------------

In your templates: `<GravatarImage @email="user@example.com" />`
